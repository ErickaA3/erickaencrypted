Promedio bueno o malo

Este ejemplo, claro que se puede automatizar con un for o while, pero nos centraremos en el if y el else

1️⃣Se asignan una variable de con lista “notas” y 5 variables para hacer enteros a los espacios que se pide que se llenen. Es un poco enredado pero te lo explico, se piden las notas al usuario con las variables separadas no1, no2.. y después esas mismas variables se utilizan para asignarle las notas en enteros a las posiciones de la lista. Después para la suma y división se utiliza las variables notas[0]..notas[4] que ya estan asignadas.

2️⃣En este ejemplo se utiliza la condición que para que el promedio sea excelente tiene que ser mayor a 90(resuladopro >=90), con el if, sino el else actúa e imprime en pantalla que se puede superar

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial- 
    scale=1.0">
    <title>ERICKA</title>
    <script>
        var notas=[0, 0, 0 ,0 ,0]
        var no1=0
        var no2=0
        var no3=0
        var no4=0
        var no5=0
        var resultadopro
        no1=window.prompt("Digite su primera nota")
        no2=window.prompt("Digite su segunda nota")
        no3=window.prompt("Digite su tercera nota")
        no4=window.prompt("Digite su cuarta nota")
        no5=window.prompt("Digite su quinta nota")
        notas[0]=parseInt(no1)
        notas[1]=parseInt(no2)
        notas[2]=parseInt(no3)
        notas[3]=parseInt(no4)
        notas[4]=parseInt(no5)
        resultadopro= notas[0]+notas[1]+notas[2]+notas[3]+notas[4]
        resultadopro=resultadopro/3
        if (resultadopro>=90) {
            document.writeln("<p> Su promedio es de: 
      "+resultadopro+"</p>"+"<p>¡Exelente!</p>")
 
        }
        else
            document.writeln("<p> Su promedio es de: 
       "+resultadopro+"</p>"+"<p>¡Puedes superarte!</p>")
    </script>
</head>
<body>
</body>
</html>
