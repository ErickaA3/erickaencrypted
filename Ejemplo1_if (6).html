Sueldo bruto

Un ejemplo de obtener el sueldo bruto dependiendo de las horas trabajadas

1️⃣4 variables, “tarifa” cuanto cobra por hora, “horas” cuantas horas trabajadas, “horasA” para asignar a “horas” lo ingresado por el usuario en entero, “total” sueldo bruto

2️⃣Este ejercicio se hizo en base a una práctica por eso las condiciones sin explicación. Si horasA es mayor a 40 se realizará la operación normal para sacar el sueldo bruto, sino se cumple el else actuará con otras especificaciones para sacar el sueldo bruto.

<!DOCTYPE html>
    <head>
        <meta charset="uft-8">
        <title>ERICKA1</title>
 
        <link rel="stylesheet" href="estilos.css">

         href="https://fonts.googleapis.com/css?family=Trirong">
 
        <Script>
            var tarifa;
            var horas;
            var total;
            var horasA
            tarifa= window.prompt("¿Cúal es su tarifa por hora?")
            horasA= window.prompt("¿Cuantas horas trabajo?")
            if(horasA <= 40){
                horas= parseInt(horasA);
                total= tarifa * horas;
                document.writeln("Su sueldo bruto es:"+ total);
            }
            else{
                horas= parseInt(horasA);
                horas - 40;
                horas * 1.5;
                total= (horas + 40) * tarifa ;
                document.writeln("Su sueldo bruto es:"+ total);
            } 
        </Script>

    </head>
