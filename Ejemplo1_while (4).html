Lista números

Un ejemplo nada complejo, perfecto para entender bien como funcionan los ciclos

1️⃣Solo se necesita 1 variable, en este caso “num”, el while tendrá de condición num<=500, puede ser 500 o cualquier otra cifra, después imprime en pantalla la variable num y aumenta 1, para hacer una cuenta simple del 1-500.

  <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script>
     var num = 0;
    while (num <= 500) 
{ 
        document.writeln(num);
 
            ++num;
}
    </script>
</head>
<body>
 
</body>
</html>
