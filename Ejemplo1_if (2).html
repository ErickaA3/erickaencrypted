Cuenta Regresiva

Este ejemplo es una cuenta Regresiva, hasta 10, los créditos de este programa a Shair por si quieres ver el video completo. De igual manera te lo explicaré

1️⃣La variable “segundoInicio”, que es desde donde comienza la cuenta Regresiva.

2️⃣la función actualizar, que tiene el programa en sí, imprime countdown, que se asigna a segundoInicio con innerHTML

3️⃣El if pide que segundo inicio sea igual a 0,si esto se cumple finaliza. Sino se cumple el else, que le quita 1,asi haciendo la cuenta Regresiva hasta 5.

4️⃣SetTimeout, es el tiempo de espera, que también hace que se vuelva a ejecutar la función.

5️⃣Importante, el código de arriba es el que irá en un archivo con extensión .js y el de abajo es html, tener en cuenta los nombres que se le pone, en este ejemplo se llamo app.

var segundoInicio = 5;
 
function actualizar() {
    document.getElementById("countdown").innerHTML = segundoInicio;
 
    if (segundoInicio == 0) {
        document.getElementById("countdown").innerHTML = "FIN";
    } else {
        //segundoInicio-=1
        segundoInicio = segundoInicio -1;
        setTimeout(actualizar, 1E3);
    }
}
 
actualizar();
